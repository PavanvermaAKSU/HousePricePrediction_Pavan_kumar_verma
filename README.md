# House Price Prediction

## Overview
This project predicts house prices using Machine Learning.

## Dataset
Housing.csv

## Models Used
- Linear Regression
- Random Forest Regressor

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Results
Linear Regression performed best with an R² Score of 0.605.

- Data Cleaning
- Feature Encoding
- Data Visualization

- House Price Prediction
